# Zozi - Companion NPC Mod for Minecraft

## Overview
Zozi is a Minecraft Forge mod that creates an AI companion NPC that:
- ✅ Spawns when you join a world
- ✅ Follows you around like a loyal companion
- ✅ Helps you in combat
- ✅ Collects resources
- ✅ Builds and manages farms
- ✅ **Completely offline** - no internet connection needed!

## Requirements
- Minecraft 1.20.1
- Forge 47.2.0+
- Java 21+

## Installation
1. Download the latest `zozi-mod.jar` from the releases
2. Place it in your `%APPDATA%\.minecraft\mods` folder (Windows)
3. Launch Minecraft with Forge
4. Zozi will automatically spawn when you join a world!

## Building from Source
```bash
git clone https://github.com/Utofarman1/Zozi-Companion-Mod.git
cd Zozi-Companion-Mod
./gradlew build
```

The compiled JAR will be in `build/libs/`
